Inherits from SChartAxis : NSObject
Declared in SChartCategoryAxis.h

Overview

A SChartCategoryAxis is a subclass of SChartAxis designed to work with discrete data points with no linear correlation between values. For the purposes of ranges, each data point value for a category axis is represented by an index (generated by the order in which data points are added to the series – but removing duplicates). This determines order and can be used to set ranges.
