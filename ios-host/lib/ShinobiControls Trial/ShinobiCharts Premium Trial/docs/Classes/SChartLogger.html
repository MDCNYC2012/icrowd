SChartLogger Class Reference

Inherits from NSObject
Declared in SChartLogger.h

Overview

A class to format output and control the level of reporting by the chart and its objects. Setting the log level to SChartLogLevelError will output all messages below this level ie: SChartLogLevelDebug and SChartLogLevelWarning.

Log Levels:
typedef enum {
SChartLogLevelDebug, // print out everything, include debug and diagnostic messages
SChartLogLevelWarning, // print out most things, including warnings
SChartLogLevelError, // only print errors and messages causing ShinobiChart to abort
SChartLogLevelSilent // print nothing at all
} SChartLogLevel;
